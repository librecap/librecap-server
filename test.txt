Changelog v0.1.0
- Initial implementation of LibreCap server with Actix web framework
- Privacy-focused CAPTCHA system using dog image recognition
- Tiered Proof of Work challenge system with configurable difficulty levels
- Image CAPTCHA generation with WebP compression and manipulation techniques
- HMAC-SHA256 cryptographic validation for challenge signing
- Redis-based session management for tracking nonces and states
- IP and User-Agent verification to bind challenges to clients
- Rate limiting through increasing PoW difficulty
- Efficient image dataset handling with built-in caching
- Docker and docker-compose configuration for easy deployment
- Production-optimized Rust binary with link-time optimization
- Comprehensive API documentation for /librecap/v1/initial and /librecap/v1/challenge endpoints
- Environment variable configuration for host, port, workers, and Redis settings
